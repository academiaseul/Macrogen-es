  Fun4me, which is wrapper tools including FragGeneScan, RAPsearch2, MinPath, facilitate prediction of protein-coding gene from short reads. The gene prediction from the reads enable to predict that what microbiome in the sample do. The predicted genes were compared with MetaCyc database for pathway analysis and EC (Enzyme Commission) number of genes are retrieved. The information of the pathway are listed as table and visualized as chart.

  Pathway analysis was conducted based on MetaCyc database and the result is listed as table. You can get informations from the table such as pathway ID, description of pathway and how many EC number are involved in the pathway. Please click MetaCyc ID of the pathways to get more information of the pathways, then, it will be redirected to MetaCyc webpage


  •     · Pathway : The matched pathway id (MetaCyc ID).
  •     · Description : The description of the matched pathway.
  •     · #Functions : The total count of enzyme in the matched pathway.
  •     · #Annotated : The total count of annotated enzyme.
  •     · Functions : The EC number of annotated enzyme.
