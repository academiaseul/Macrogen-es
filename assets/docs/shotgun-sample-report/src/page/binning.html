    The unclassified contigs from the Centrifuge results were binned using unsupervised binning methods using CONCOCT v1.1.0. The contigs belong to a same bin have similar characteristics of sequences. The contigs were classified by charateristics of sequences, not taxon, therefore, the contigs in a bin could be originated from different species which have similar sequence characteristics on some part of their genome.
