Show Description

  The number and percentage of taxon classified from reads were calculated to display contents of species in the sample. You can see the results from phylum to subspecies level by clicking taxon level you want to see. Taxonomy name "k__;c__;o__;f__;g__;s__;n__" means "Root" which is highest level of the taxon. Root can be detected when reads have multiple matches with different superkingdom species. The reads which is not classified are described as "Unclassified". Abundance were normalized value based on normilization formula. For the detail, please refer to the Taxonomy assignment for reads section of Methods page. Abundance can be calculated in genus, species and subspecies level, therefore, abundance of the others is described as NA.
