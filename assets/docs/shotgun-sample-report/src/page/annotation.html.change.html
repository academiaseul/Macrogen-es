  To understand what members of the meta-genome do, genes of the meta-genome were predicted and annotated. Prokka were used for gene prediction. Predicted genes were annotated using eggNOG-mapper 2.0.1, which is diamond based annotation tools, with EggNOG database v5. EggNOG database is a hierarchical, functionally and phylogenetically annotated orthologoy resource. Similar with COG database, genes in the database are categorized as 25 categories.


  •     · Total Predicted Protein : The number of predicted proteins
  •     · EggNOG Matched Proteins : The number of proteins matched with EggNOG database
  •     · Single EggNOG category : The number of proteins having single EggNOG category
  •     · Multiple EggNOG category : The number of proteins having multiple EggNOG category
  •     · Unmatched Proteins : The number of proteins not matched with EggNOG database
