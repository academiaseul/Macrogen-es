    How to Customize Samples:

    1. Click Download Template to get the metadata.xlsx file.
    2. Open the file and specify your custom sample order in the 'SampleID' column.
    3. Save the changes and click Upload File to apply your custom ordering.

    Important: Custom sample settings are temporary. If you navigate away or refresh this page, you will need to re-upload the file.
