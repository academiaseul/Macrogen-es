    Diversity analysis

    (Beta diversity > UPGMA tree)


    UPGMA Tree - Bray Curtis
    Select a Group Type:


    * Unweighted Pair Group Method with Arithmetic mean (UPGMA) is type of hierarchical clustering method using average linkage and can be used to interpret the distance matrix (bray_curtis_upgma_cluster.tre). The output is a file that can be opened with tree viewing software, such as FigTree. For your own purposes, the distance matrix file can be used to draw your own phylogenetic tree using a suitable tool

    Group Management

    How to Customize Groups:

    1. Click Download Template to get the metadata.xlsx file.
    2. Open the file and specify your custom groups for each sample in the 'Group' sheet.
    3. Save the changes and click Upload File to apply your custom grouping.

    Important: Custom group settings are temporary. If you navigate away or refresh this page, you will need to re-upload the file.
